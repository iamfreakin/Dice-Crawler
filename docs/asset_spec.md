# 에셋 스펙 (픽셀아트) — Dice Crawler

> **화풍 확정: 픽셀아트.** 원본은 작게 그리고 게임에서 정수배(2x·3x·4x) 확대한다.
> 프로젝트 텍스처 필터는 Nearest로 설정됨(`default_texture_filter=0`) → 확대해도 픽셀이 또렷.

## 공통 규칙
- 경로: `assets/sprites/<종류>/<파일명>.png` — **파일명 = 코드 id** (소문자 영문, 정확히 일치)
- 형식: PNG, **투명 배경(알파)**. 배경 이미지만 불투명
- **안티에일리어싱 끄기**(계단 픽셀 그대로), **1px 깔끔한 외곽선**, 제한된 팔레트
- 한 픽셀 = 1단위로 그릴 것. 원본을 미리 키워서 그리지 말 것(흐려짐)
- 게임 팔레트: 배경 `#16161a`, 패널 `#1f1f27`, 보라 액센트 `#7f77dd`
- 속성 색: 화염 `#e2503a` / 빙결 `#37c4dd` / 번개 `#efc127` / 저주 `#7a3aa0`
- 등급 색: 일반 `#b4b2a9` / 레어 `#378add` / 에픽 `#7f77dd` / 유니크 `#ef9f27` / 레전더리 `#e24b4a`

## 네이티브 크기 요약 (이게 기준!)
| 종류 | 원본(네이티브) px | 게임 표시 배율 |
|------|------------------|---------------|
| 일반 적(고블린/박쥐) | **48×48** | 3~4x |
| 엘리트(오크) | **64×64** | 3~4x |
| 보스(드래곤) | **96×96** | 3x |
| 의도/면 아이콘 | **16×16** (또는 32×32) | 2~3x |
| 유물 아이콘 | **32×32** | 2x |
| 주사위 몸체 | **32×32** | 3x |
| 배경 | **320×180** | 4x → 1280×720 |

## ① 폰트 — `assets/fonts/main.ttf` (최우선)
- **한글 지원 + 픽셀 폰트** 추천: 갈무리11, 둠스데이, 네오둥근모, Galmuri
- 픽셀 폰트는 특정 크기(예: 11/16/22px)에서만 또렷 → 사용 크기 알려주면 거기 맞춰 세팅
- ⚠️ 이모지는 폰트에 없음 → ④⑤⑥ 아이콘으로 대체

## ② 적 — `assets/sprites/enemies/`
구도 공통: 정면/3·4 측면, **발이 캔버스 하단 중앙**, 머리 위 2~4px 여백. 외곽 1px 어두운 라인.

### `goblin.png` (48×48)
- 작고 비열한 잡몹. 녹색 피부, 누더기, 녹슨 단검. 구부정·음흉. 위협보다 "성가신" 톤.

### `cave_bat.png` (48×48, 몸은 더 작게)
- 졸개. 날개 편 박쥐, 공중에 뜬 포즈(중앙 정렬). 어두운 보라 몸 + 빨간 눈 2px 포인트.

### `orc_elite.png` (64×64)
- 덩치 큰 엘리트. 고블린보다 큰 실루엣, 근육·갑옷 일부, 큰 도끼. 어깨 벌린 정면.
- 엘리트 표식: 외곽에 보라(#7f77dd) 1px 글로우.

### `dragon_boss.png` (96×96)
- 보스. 날개·뿔·긴 목, 입가 불씨. 짙은 비늘 + 가장자리 적색(#e24b4a) 발광. 캔버스 거의 꽉 차게.

## ③ 유물 아이콘 — `assets/sprites/relics/` (32×32)
정사각 꽉 찬 단일 오브젝트(뱃지 느낌), 가운데 정렬, 외곽 1px 라인.

- `reroll_charm.png` — 회전 화살표(↻) 부적/메달, 레어 파랑(#378add)
- `steel_scale.png` — 금속 비늘/작은 방패, 은색 + 파란 반사
- `ember_core.png` — 떠다니는 불씨 코어, 주황빨강(#e2503a) 중심광
- `life_rune.png` — 룬 새겨진 붉은 보석/심장형 돌, 적색 글로우

## ④ 적 의도 아이콘 — `assets/sprites/intents/` (16×16)

- `reinforce.png` — 방패(강화), 파랑
- `summon.png` — 깃발/소환진(증원), 보라

## 작업 순서 & 폴백
1. `main.ttf`(한글 픽셀 폰트) → 2. 적 4종 → 3. 의도·면·유물 아이콘 → 4. 배경/주사위 몸체

**일부만 먼저 줘도 됨.** 없는 슬롯은 텍스트/이모지로 자동 폴백. `assets/`에 넣고 알려주면 슬롯 연결.

# Dice Crawler — 로드맵 / 마일스톤

> 작업 우선순위와 목표 분량. 기획 의도는 [design.md](design.md), 진행 체크리스트는 루트 `CLAUDE.md` "현재 상태".

## 전략: "항상 플레이 가능한 완성품"을 유지하며 키운다

대량 콘텐츠(적 12~18, 유물 30~50)를 한 번에 목표하면 지친다.
중간에 **출시 가능한 수직 슬라이스(1막)**를 끼워 완성도부터 올린 뒤, 양을 확장한다.

---

## 작업 순서 (현재 → 출시)

### Phase 1 — 토대: 롤 컨텍스트 + 계산 분리 `[다음]`
- 정산을 **순수 계산 `preview(롤컨텍스트, 타겟) → {피해, 방어, 화상, 시너지...}`** 와 **적용**으로 분리.
- ⚠️ 단순 함수가 아니라 **롤 컨텍스트**로: 순서 정렬된 결과 + 다음 굴림 보정 스택 + 이월 상태
  (현재 순서 없는 `_rolled` dict로는 연쇄/직전결과/예열 불가).
- **이벤트 로그 + replay**(리롤 옵션2): **rng_token 저장**, 효과만 재적용, 리롤=토큰 1개 재발급 후 재생.
  상태 4계층 분리 / 무한 콤보 가드 — 상세는 **design.md "롤 컨텍스트" 절**이 정본.
- 미리보기·면 조작·연쇄·조건부·밸런스 **전부의 토대**.

### Phase 2 — 정보 전달(가독성)
- 굴리는 중 **예상 피해/방어 실시간 표시** (Phase 1의 preview 사용)
- 적 의도 **결과 미리 계산**("막고 나면 HP 얼마")
- 상태이상 **툴팁**
- 로그를 안 읽어도 상황 이해 가능하게.

### Phase 3 — 정체성: 면 조각 + 조작/연쇄 ⭐
- `[완료]` **Phase 3-A**: `FaceData(tags + effects)` + `FaceEffectData(timing)` +
  순수 `EffectResolver` 도입, 기존 면 전체 데이터 이관(전투 동작 보존).
- `[다음]` **Phase 3-B**: 조작 면을 **증폭 → 예열 → 변환 → 복제 → 보존** 순서로 추가.
- `[이후]` **Phase 3-C**: "주사위 선택 → 면 선택 → 적용" 면 조각 UI.
- 조각 모디파이어 `ENHANCE / REPLACE`(EXPAND는 후순위) + "주사위 선택 → 면 선택 → 적용" UI(보상/상점/휴식 공유)
- 면 효과를 데이터/전략 + `timing` 필드로 분리(enum+분기 탈피)
- **MVP 특수 면(이 순서로 구현)**: **증폭 → 예열 → 변환 → 복제 → 보존**
  (앞 3개는 단순 수정/pending, 복제·보존은 출처추적·턴간상태라 뒤로)
- 분할 보류. 연쇄(촉매/반향/마무리/연결)·조건부는 MVP 직후 2차 웨이브

### Phase 3.5 — 적 주사위 디버프 (양념)
- **정화 수단 먼저 구현** → 그 뒤 **빙결·봉인·과부하 3종** (4원칙: 복구·예고·예측·정화가 코드로 강제)
- 영구 저주 면 주입은 엘리트·이벤트 보상에만

### Phase 4 — 손맛(juice)
- 선행: 전투 UI를 **전체 재생성 → 노드 갱신** 구조로 리팩토링
- 주사위 굴림 애니, 타격/화상/방어 연출, 화면 흔들림, 데미지 숫자 팝업, 적 사망 연출
- 효과음 + BGM

### Phase 5 — 1막 콘텐츠 채우기 (아이콘 아트 병행)
- 아래 "1막 수직 슬라이스" 분량까지 .tres + 아이콘으로 확장
- 이벤트 노드, 2~3 빌드 방향 성립

### Phase 6 — 출시 기능
- 중간 저장 / 이어하기 (Phase 1~ 에서 직렬화 가능한 상태 유지 전제)
- 설정(음량/해상도/전체화면), 일시정지/재시작/종료 확인
- 튜토리얼, 키보드/패드 조작, 엔딩 + 크레딧

## 콘텐츠 목표

### 1막 수직 슬라이스 (먼저 완성할 마일스톤)
- 적 **8종** / 엘리트 **2** / 보스 **1**
- 유물 **~15** / 특수 면 **~10**
- 빌드 방향 **2~3개** 성립
- 첫 클리어 **40~60분**, 완성도(시스템·연출·UI) 높게

### 상업 출시 권장 최소치 (확장 목표)
- 적 **12~18종** / 엘리트 **4+** / 보스 **3**
- 유물 **30~50종** / 특수 면 **15~25종**
- 이벤트 노드 **10+종**
- 빌드 방향 **2~4개**
- 첫 클리어 **2~4시간**

## 현재 보유 (데모)
- 주사위 3 (공격/방어/스킬), 적 4 (고블린/박쥐/오크정예/드래곤보스), 유물 4
- 시스템: 에너지 적응형 전투, 개별 리롤, 상태이상, 시너지, 다중 적/타겟, 상점/골드, 분기 맵
- 비주얼: 픽셀 에셋 연결, 폰트, HP바, 패널, 맵 연결선

→ **시스템은 거의 갖춰졌고, "면 조각(정체성) + 정보 전달 + 손맛 + 콘텐츠"가 남은 핵심.**

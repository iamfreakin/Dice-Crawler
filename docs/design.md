# 🎲 Dice Crawler — 게임 기획

턴제 주사위 로그라이크. Godot 4 / GDScript / PC(itch.io 우선) / 솔로 개발.

## 핵심 루프

1. **노드 맵 탐색** — 분기 선택 (일반 전투 / 엘리트 / 휴식 / 상점 / 보스)
2. **적 의도 확인** — 스테이지 진입 전 적 구성 및 의도 아이콘 미리 공개
3. **전투 — 주사위 드로우 & 굴림** — 풀에서 랜덤 드로우 → 핸드에서 2개 선택 → 굴림 → 눈으로 행동 확정
4. **리롤 토큰 관리** — 스테이지당 지급, 미사용 시 소멸. 재굴림 여부 자원 관리
5. **보상 획득** — 등급별 랜덤 보상 (새 주사위 / 면 업그레이드 / 유물)
6. **반복 → 보스** — 빌드 완성하며 층 클리어, 최종 보스 처치

## 주사위 시스템

### 기본 주사위 3종

| 주사위 | 기본 면 | 효과 |
|--------|---------|------|
| ⚔️ 공격 | 1~6 균등 | 데미지 / 탄약 수 |
| 🛡️ 방어 | 1~6 균등 | 체력 / 실드 |
| ✨ 스킬 | 🔥 ❄️ ⚡ 💀 🔄 + 숫자 | 특수 효과 발동 |

### 면 커스터마이징 (핵심 빌드업)

런 중 보상으로 주사위 면을 직접 조각한다.

- **면 교체**: 숫자 면 → 특수 효과 면으로 변경
- **면 강화**: 숫자 값 상향 (예: 3 → 5)
- **주사위 확장**: 6면 → 8면, 더 많은 효과 탑재

### 드로우 구조

주사위 풀(덱)에서 랜덤 드로우 → 핸드 구성 → 풀 소진 시 셔플 후 재드로우 (StS 카드 드로우와 동일 구조).

## 전투 시스템

### 적 의도 시스템

- **일반 적** — 의도 전부 공개
- **엘리트** — 의도 일부만 공개
- **보스** — 페이즈 패턴만 공개

의도 아이콘:

- ⚔️ 돌격 — 근접 러시
- 🎯 저격 — 원거리 고데미지
- 💣 폭발 — 광역 공격
- 🛡️ 강화 — 방어력 상승
- 📡 증원 — 추가 적 소환

### 리롤 토큰

- 스테이지당 기본 1~2개 지급
- 미사용 시 다음 스테이지로 이월 안 됨
- 저주 면 굴릴 때 보너스 지급

### 시너지 보너스

같은 속성 주사위 2개 이상 굴리면 추가 효과 — 예: 🔥 두 개 → 폭발 데미지.

## 보상 등급

| 등급 | 확률 | 효과 |
|------|------|------|
| 일반 | 50% | 숫자 면 소폭 강화 |
| 레어 | 28% | 특수 효과 면 추가 |
| 에픽 | 15% | 주사위 자체 교체 / 확장 |
| 유니크 | 5% | 런 전체를 바꾸는 유물 |
| 레전더리 | 2% | 면 전체 재조각 + 저주 딸려옴 |

- 보스 클리어 → 레어 이상 보장
- 엘리트 클리어 → 에픽 이상 보장

## 기술 스택

- Godot 4 + GDScript
- `GameManager` Autoload 로 런 상태 전역 관리
- `DiceData` / `EnemyData` / `IntentData` 를 Resource 로 관리
- `BattleManager` 상태머신 (`DRAW → PLAYER_TURN → RESOLVE → ENEMY_TURN → REWARD`)
